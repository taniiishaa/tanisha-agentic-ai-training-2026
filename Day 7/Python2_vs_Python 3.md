# Python 2 vs Python 3

## Overview
Python 2 and Python 3 are two major versions of the Python programming language. Python 3 is the latest and recommended version, while Python 2 is no longer supported.

## Comparison

| Feature | Python 2 | Python 3 |
|----------|----------|----------|
| Print Syntax | `print "Hello"` | `print("Hello")` |
| Division | `5/2 = 2` | `5/2 = 2.5` |
| User Input | `raw_input()` | `input()` |
| String Type | ASCII by default | Unicode by default |
| Range Function | Returns a list | Returns an iterator |
| Support Status | Ended in 2020 | Actively maintained |

## Example

### Python 2
```python
print "Hello World"
name = raw_input("Enter your name: ")
```

### Python 3
```python
print("Hello World")
name = input("Enter your name: ")
```

## Why Choose Python 3?

- Improved performance
- Better memory management
- Built-in Unicode support
- Cleaner syntax
- Supported by modern libraries and frameworks

## Conclusion

Python 3 is the preferred choice for modern software development. It offers better features, improved security, and ongoing support, making it suitable for all new projects.
